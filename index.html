Storage Resource Management

Sebuah website sederhana untuk Storage Resource Management yang mengandalkan Google Spreadsheets API sebagai database-nya. Dengan desain statis, website ini menyajikan informasi terkini mengenai pengelolaan sumber daya penyimpanan dengan memanfaatkan integrasi yang efisien dengan Google Spreadsheets API sebagai basis data utamanya.
